Задание: PARTS (компьютерные комплектующие)
Реализовать простенькое приложение Parts-list, для отображения списка деталей для сборки компьютеров на складе.

Шаги для запуска:
1. Запустить SQL скрипт для создания и заполнения БД createParts.sql
2. После деплоя в TomCat главная страница приложения находится по адресу: http://localhost:8080/parts
